# react-native-code-editor

a code editor component for react native

## Installation

```sh
npm install react-native-code-editor
```

## Usage

```js
<CodeEditor language={"Swift"} theme={"atom-one-dark"} style={{backgroundColor: "#000000", height: 300, width: 200}} />
```
